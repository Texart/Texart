Class TxReferenceScheme

A TxReferenceScheme represents a URI scheme (such as file: and https:). As such, it is also case-insensitive.

Note that not every valid URI scheme is a valid TxReferenceScheme. Specifically, the characters, +, and ., are not allowed. However, - is allowed. This restriction may be relaxed in the future.

Refer to https://tools.ietf.org/html/rfc3986#section-3.1 for information on URI schemes (which are valid TxReferenceSchemes, except for the restrictions listed above).

//    scheme      = ALPHA *( ALPHA / DIGIT / "+" / "-" / "." )

Use cases:

  • As a way to allow custom resolution strategies for #load and #r directives in Texart scripts.
  • As a part of locating resources in a ITxPlugin assembly. See TxPluginResourceLocator.

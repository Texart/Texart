Class TxPluginResource<T>

A union type which is used as the return type of ITxPlugin lookup methods. This is required because C# (as of 8.0) does not support union types. This type allows the return value to be flexible. See TxPluginResource members for more details on the semantics of each kind of member. TxPluginResource<T> is immutable.

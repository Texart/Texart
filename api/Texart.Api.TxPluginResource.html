OfFactory<T>(TxFactory<T, TxArguments>)

Creates a TxPluginResource<T> with ActiveMemberKind set to Factory and Factory set to factory.

The factory function accepts a TxArguments which can be used configure the created instance. The format of the arguments is implementation-defined.

Redirect<T>(TxPluginResourceLocator, Func<TxArguments, TxArguments>)

Creates a TxPluginResource<T> with ActiveMemberKind set to Redirect and Redirect set to a TxPluginResource.ResourceRedirect with locator and argumentsTransformer.

The locator is an absolute URI identifying the location where a plugin believes the demanded resource can be found. This includes a TxReferenceScheme, AssemblyPath, and ResourcePath.

Widen<T, TU>(TxPluginResource<TU>)

Widens the type TU to T. TxPluginResource<T> can actually be contravariant. However, C# (as of 8.0) does not support contravariant generic type parameters for classes. The returned value and resource share the same ActiveMember.
